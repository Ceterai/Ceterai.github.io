# Alta Washer

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/objects/alta/basic/washer/icon.png" alt="Alta Washer icon" loading="lazy" width="auto" height="16px"/> **Alta Washer** is a common storage unit.

A simple compact titanium washing machine. Not a lot of clothes can fit inside.  
Its design is quite similar to most washing machines, and is quite careful with silk and cotton.

## Usage

### Storage

Can store items, slot count is outlined in parameters. Does **NOT** preserve food.
