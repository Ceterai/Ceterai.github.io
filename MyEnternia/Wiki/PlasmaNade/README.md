# Plasma Nade

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/items/throwables/ct_plasma_nade.png" alt="Plasma Nade icon" loading="lazy" height="16px" width="auto" /> **Plasma Nade** is a rare electric throwable item.

A blast device that bursts with [plasma](https://ceterai.github.io/MyEnternia/Wiki/Tags/Plasma) on impact. Provides lasting damage from blobs of plasma.
