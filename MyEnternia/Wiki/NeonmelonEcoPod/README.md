# Neonmelon Eco Pod ★★★

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/objects/farmables/alta/ground/produce/neonmelon/pod/icon.png" alt="Neonmelon Eco Pod ★★★ icon" loading="lazy" width="auto" height="16px"/> **Neonmelon Eco Pod ★★★** is a legendary plantable item.

Contains a modified neonmelon sapling, as well as all things necessary for it to ripe. Can be placed anywhere.
