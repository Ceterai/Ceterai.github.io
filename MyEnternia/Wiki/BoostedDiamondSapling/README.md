# Boosted Diamond Sapling ★★

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/objects/farmables/alta/liquid/diamond/boosted/icon.png" alt="Boosted Diamond Sapling ★★ icon" loading="lazy" height="16px" width="auto" /> **Boosted Diamond Sapling ★★** is a legendary plantable item.

This sapling comes with enhanced mutations that allow much faster ripening. Requires to be at least 50% in a liquid.
