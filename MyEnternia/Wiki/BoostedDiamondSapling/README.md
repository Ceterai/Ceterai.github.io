# Boosted Diamond Sapling ★★

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/objects/farmables/alta/liquid/diamond/boosted/icon.png" alt="Boosted Diamond Sapling ★★ icon" loading="lazy" width="auto" height="16px"> **Boosted Diamond Sapling ★★** is a legendary plantable item.

This sapling comes with enhanced mutations that allow much faster ripening. Requires to be at least 50% in a liquid.

## Ingame

Species descriptions:

- Alta: A diamond bud! It will take a long time, but it will grow into more diamonds!

Growing stages:

2. Lasts from 280 to 320 seconds.
3. Lasts from 1170 to 1230 seconds.
