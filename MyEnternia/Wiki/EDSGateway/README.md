# EDS Gateway

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/objects/alta/eds/decorative/gateway/icon.png" alt="EDS Gateway icon" loading="lazy" height="16px" width="auto" /> **EDS Gateway** is an uncommon door.

A reinforced vertical hatch with dark red lines all over it. A powerful blast door made by [EDS](https://ceterai.github.io/MyEnternia/Wiki/Tags/Eds).
