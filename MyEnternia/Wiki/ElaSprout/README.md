# Ela Sprout

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/objects/farmables/alta/liquid/ela/icon.png" alt="Ela Sprout icon" loading="lazy" height="16px" width="auto" /> **Ela Sprout** is an uncommon plantable item.

A pretty glowing plant full of colorful vines. Requires to be at least 50% in a liquid.

## Ingame

Species descriptions:

- Alta: Ela plants usually can only be found close to the cores of enterash planets.

Radiomessages on pickup:

- I see that you've discovered some seeds. I suggest planting them in some tilled soil, and watering them until they grow. The tools you need can be built at a foraging table.
