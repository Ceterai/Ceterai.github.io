# Gyrocore

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/items/throwables/ct_alta_testing_ball.png" alt="Gyrocore icon" loading="lazy" height="16px" width="auto" /> **Gyrocore** is an uncommon throwable item.

A large heavy metal ball with crystal panels and energy lines. Can be used for testing or as part of big mechanisms.
