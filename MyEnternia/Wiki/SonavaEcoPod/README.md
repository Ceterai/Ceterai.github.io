# Sonava Eco Pod ★

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/objects/farmables/alta/ground/sona/pod/icon.png" alt="Sonava Eco Pod ★ icon" loading="lazy" height="16px" width="auto" /> **Sonava Eco Pod ★** is a legendary plantable item.

Contains a modified sonava sapling, as well as all things necessary for it to ripe. Can be placed anywhere.

## Ingame

Species descriptions:

- Alta: This plant produces snow-like substance that is basically plant-based snow!
