# Izo Jam

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/items/generic/food/tier1/ct_iso_jam.png" alt="Izo Jam icon" loading="lazy" height=16px width="auto" /> **Izo Jam** is a common dish.

A great way to prepare nia treats is with a good amount of cold isoslime jam. It's also really useful for making ice cream!  
It's cooling capabilities make it a great ingredient in almost any cold dish, really.

## Ingame

Species descriptions:

- Alta: Slimey, but so sweet and nice, and overall pretty chill.
