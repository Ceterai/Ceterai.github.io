# Alta Shake

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/items/generic/food/tier3/ct_tonna_shake.png" alt="Alta Shake icon" loading="lazy" width="auto" height="16px"/> **Alta Shake** is an uncommon drink.

A simple alta milkshake made with most basic ingredients available in alta culinary culture.  
While its components are pretty common to find, it doesn't impact its taste negatively in any way.

## Ingame
