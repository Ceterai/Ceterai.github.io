# Eco Evara Sapling ★

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/objects/farmables/alta/ground/eva/eco/icon.png" alt="Eco Evara Sapling ★ icon" loading="lazy" height=16px width="auto" /> **Eco Evara Sapling ★** is an uncommon plantable item.

This cloud fibre sapling was genetically modified to fit into small spaces and consume almost no liquids.

## Ingame

Species descriptions:

- Alta: Evara plants usually grow in haven gardens, even though you'd expect to see them grow on top of eva clouds.
