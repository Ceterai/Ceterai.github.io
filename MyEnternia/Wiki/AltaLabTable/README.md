# Alta Lab Table

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/objects/alta/lab/table/icon.png" alt="Alta Lab Table icon" loading="lazy" height="16px" width="auto" /> **Alta Lab Table** is an uncommon piece of furniture.
