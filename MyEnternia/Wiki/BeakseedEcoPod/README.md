# Beakseed Eco Pod ★

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/objects/farmables/alta/ground/produce/beakseed/pod/icon.png" alt="Beakseed Eco Pod ★ icon" loading="lazy" width="auto" height="16px"/> **Beakseed Eco Pod ★** is a legendary plantable item.

Contains a modified beakseed sapling, as well as all things necessary for it to ripe. Can be placed anywhere.

## Ingame

Species descriptions:

2. Lasts from 280 to 320 seconds.
3. Lasts from 280 to 320 seconds.
4. Lasts from 1860 to 1940 seconds.
5. Can be harvested:
`beakseedHarvest`
