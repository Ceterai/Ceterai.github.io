# Random Monster Spawn Egg

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/items/active/alta/spawners/ct_monster_spawner.png" alt="Random Monster Spawn Egg icon" loading="lazy" width="auto" height="16px"/> **Random Monster Spawn Egg** is a legendary one-handed throwable item.

Throw it down to release the creature captured inside! Contains a random creature from the My Enternia mod.
