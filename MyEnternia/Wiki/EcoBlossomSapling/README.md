# Eco Blossom Sapling

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/objects/farmables/alta/ground/blossom/eco/icon.png" alt="Eco Blossom Sapling icon" loading="lazy" width="auto" height="16px"/> **Eco Blossom Sapling** is a rare plantable item.

This silk fibre sapling was genetically modified to fit into small spaces and consume almost no liquids.
