# Bionid Glowstick

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/items/active/alta/glowsticks/bionid.png" alt="Bionid Glowstick icon" loading="lazy" width="auto" height="16px"/> **Bionid Glowstick** is a rare throwable item.

Creates a bright bionic glow when thrown. Is made from rare biocrystallic materials.  
Has to come from bionid specifically, since that's the most luminescent type of bionid.

## Ingame

Species descriptions:

- Alta: The color of my plasma... Eery, but interesting.
