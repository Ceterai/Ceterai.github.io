# Wild Aya Tuber

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/objects/farmables/alta/main/aya/icon.png" alt="Wild Aya Tuber icon" loading="lazy" height="16px" width="auto" /> **Wild Aya Tuber** is a common plantable item.

A natural source of sugar, again.

## Ingame

Species descriptions:

- Alta: A wild aya spring. Seems like this one won't grow into a tall ayaka, like usual.

Radiomessages on pickup:

- I see that you've discovered some seeds. I suggest planting them in some tilled soil, and watering them until they grow. The tools you need can be built at a foraging table.
