# Aya Tuber

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/objects/farmables/alta/main/aya/seed/icon.png" alt="Aya Tuber icon" loading="lazy" width="auto" height="16px"> **Aya Tuber** is a common plantable item.

A domesticated aya tuber, ready to grow into a small ayaka tree spring.

## Ingame

Species descriptions:

- Alta: A lab-produced aya spring. Seems like this one won't grow into a tall ayaka, like usual.

Radiomessages on pickup:

- I see that you've discovered some seeds. I suggest planting them in some tilled soil, and watering them until they grow. The tools you need can be built at a foraging table.

2. Lasts from 600 to 800 seconds.
3. Lasts from 1600 to 1800 seconds.
