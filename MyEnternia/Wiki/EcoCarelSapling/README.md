# Eco Carel Sapling

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/objects/farmables/alta/ground/produce/carel/eco/icon.png" alt="Eco Carel Sapling icon" loading="lazy" width="auto" height="16px"/> **Eco Carel Sapling** is an uncommon plantable item.

This smaller sapling will ripen much slower, but doesn't consume soil.
