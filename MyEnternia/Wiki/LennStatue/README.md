# Lenn Statue

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/objects/alta/special/secret/lenn/statue/icon.png" alt="Lenn Statue icon" loading="lazy" width="auto" height="16px"/> **Lenn Statue** is an essential decorative object.

A holographic statue on a composite platform. The label simply reads 'Lenn'.  
The shimmering static 3D image showcases what seems to be a being from another realm.
