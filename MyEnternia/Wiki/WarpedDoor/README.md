# Warped Door

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/objects/biome/alterash/warped/decorative/door/icon.png" alt="Warped Door icon" loading="lazy" width="auto" height="16px"/> **Warped Door** is a common door.

A vine door that slightly glows in the dark. The bulb in the middle seems to be its own plant.
