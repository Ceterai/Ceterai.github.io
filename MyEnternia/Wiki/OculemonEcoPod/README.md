# Oculemon Eco Pod ★★

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/objects/farmables/alta/ground/produce/oculemon/pod/icon.png" alt="Oculemon Eco Pod ★★ icon" loading="lazy" width="auto" height="16px"/> **Oculemon Eco Pod ★★** is a legendary plantable item.

Contains a modified oculemon sapling, as well as all things necessary for it to ripe. Can be placed anywhere.

## Ingame

Species descriptions:
