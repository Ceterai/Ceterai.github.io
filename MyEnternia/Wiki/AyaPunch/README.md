# Aya Punch

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/items/generic/food/tier3/ct_aya_punch.png" alt="Aya Punch icon" loading="lazy" height="16px" width="auto" /> **Aya Punch** is a rare drink.

 A sweet aya punch. Simple, but powerful.  
While the preparation process takes time, ayas sweeten out quite fast and thus are very good for a fast-marinaded punch, while most other alta punches use the fast-boiled kompot-like methods.

## Ingame

Species descriptions:

- Alta: Ayas are pretty sweet as it is, so having a less-sweet drink with a similar taste is pretty neat.
