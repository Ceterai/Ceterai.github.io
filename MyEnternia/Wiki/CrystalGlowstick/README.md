# Crystal Glowstick

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/items/active/alta/glowsticks/crystal.png" alt="Crystal Glowstick icon" loading="lazy" width="auto" height="16px"/> **Crystal Glowstick** is a common throwable item.

Creates a regular crystal glow when thrown. Is made from rare crystallic materials.  
Actual crystal are used in combination with proper glowfibre to make the effect work.

## Ingame

Species descriptions:

- Alta: The most comforting light color out there.
