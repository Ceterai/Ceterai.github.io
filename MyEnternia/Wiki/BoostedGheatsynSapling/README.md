# Boosted Gheatsyn Sapling

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/objects/farmables/alta/liquid/gheatsyn/boosted/icon.png" alt="Boosted Gheatsyn Sapling icon" loading="lazy" height="16px" width="auto" /> **Boosted Gheatsyn Sapling** is a legendary plantable item.

This sapling comes with enhanced mutations that allow much faster ripening. Requires to be at least 50% in a liquid.

## Ingame

Species descriptions:

- Alta: A gheatsyn bud! It will take a long time, but it will grow into more gheatsyn!
