# Alta Ship Bed

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/objects/alta/ship/bed/icon.png" alt="Alta Ship Bed icon" loading="lazy" width="auto" height="16px"/> **Alta Ship Bed** is a rare piece of furniture.

These cosy beds can stack on top of each other.

## Ingame

- Human: A bunk bed. Not terribly spacious, but surprisingly comfortable.
