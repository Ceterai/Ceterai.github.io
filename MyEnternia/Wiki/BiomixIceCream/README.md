# Biomix Ice Cream

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/items/generic/food/tier2/ct_biomix_icecream.png" alt="Biomix Ice Cream icon" loading="lazy" width="auto" height="16px"/> **Biomix Ice Cream** is an uncommon dish.

This icecream flavor is a refreshing mix of crystal-, bio- and herb-based ingredients.  
The recipe includes separate mixes from different cuisines, that are then piled on top of each other, resulting in a layered ice cream.

## Ingame
