# Alta Tiller

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/items/active/alta/tools/till/tiller/icon.png" alt="Alta Tiller icon" loading="lazy" height="16px" width="auto" /> **Alta Tiller** is an uncommon two-handed tool.

An advanced tool to till soil. RMB for more accurate tilling.
