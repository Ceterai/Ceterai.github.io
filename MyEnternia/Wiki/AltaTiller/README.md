# Alta Tiller

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/items/active/alta/tools/till/tiller/icon.png" alt="Alta Tiller icon" loading="lazy" height="16px" width="auto" /> **Alta Tiller** is an uncommon two-handed tool.

An advanced tool to till soil. RMB for more accurate tilling.

Can be upgraded to <img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/items/active/alta/tools/till/tiller/icon_upg.png" alt="Startiller ★ icon" loading="lazy" height="16px" width="auto" /> [Startiller ★](https://ceterai.github.io/MyEnternia/Wiki/Startiller) at the Weapon Upgrade Anvil.
