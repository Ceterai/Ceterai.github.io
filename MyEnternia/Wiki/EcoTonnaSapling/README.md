# Eco Tonna Sapling

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/objects/farmables/alta/liquid/tonna/eco/icon.png" alt="Eco Tonna Sapling icon" loading="lazy" width="auto" height="16px"/> **Eco Tonna Sapling** is a rare plantable item.

This tonna sapling was genetically modified to fit into small spaces and consume almost no liquids.
