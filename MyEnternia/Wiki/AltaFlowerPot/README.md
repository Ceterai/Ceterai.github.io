# Alta Flower Pot

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/objects/alta/special/tools/pots/alta/icon.png" alt="Alta Flower Pot icon" loading="lazy" height="16px" width="auto" /> **Alta Flower Pot** is an uncommon storage unit.

A simple flower pot made from a layered material for flowers that like more space.  
Can be used for capturing plant critters for collection and to have them as pets.
