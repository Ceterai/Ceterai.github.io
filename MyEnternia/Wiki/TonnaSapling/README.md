# Tonna Sapling

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/objects/farmables/alta/liquid/tonna/sapling/icon.png" alt="Tonna Sapling icon" loading="lazy" width="auto" height="16px"/> **Tonna Sapling** is an uncommon plantable item.

A proper sapling that takes less space and has increased and more consistent speed of growth stages. Requires to be at least 50% in a liquid.

## Ingame

Species descriptions:

- Alta: Now I don't have to climb a tonnova to get me some tonnas!

## Parameters

Growing stages:

1. Lasts from 380 to 420 seconds.
2. Lasts from 380 to 420 seconds.
3. Lasts from 1600 to 1800 seconds.
4. Can be harvested:
   - for threat level **0+** (pool rounds: x*1* (weight: 0.9), x*2* (weight: 0.1)):
