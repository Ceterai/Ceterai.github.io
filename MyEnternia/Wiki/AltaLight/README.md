# Alta Light

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/objects/alta/basic/light/icon.png" alt="Alta Light icon" loading="lazy" height="16px" width="auto" /> **Alta Light** is a common light source.

A titanium light fixture on simplistic holders. Has crystals instead of lightbulbs.  
Most basic alta furniture is designed to be simple, lightweight, and, most importantly, easy to clean and disassemble. The goal is to be comfortable and portable at the same time, while also being easy to recycle if needed.
