# Alta Grass Nade

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/items/active/alta/tools/plant/nade/ct_alta_grass_nade.png" alt="Alta Grass Nade icon" loading="lazy" width="auto" height="16px"/> **Alta Grass Nade** is a legendary throwable item.

Toss these on any ground to explode with various grass.

## Usage

### Growing Trays

Can be used as a fertilizer in <img src="https://images.steamusercontent.com/ugc/1693903078379188066/4E09EDBE233D5AA8FE9EA0EC1256E95CA3D446FF/" alt="icon" width="16" height="12"/> [Growing Trays](https://steamcommunity.com/sharedfiles/filedetails/?id=2574257768) (if you have them installed) skipping 300s of grow time with a 80% chance to be consumed.
