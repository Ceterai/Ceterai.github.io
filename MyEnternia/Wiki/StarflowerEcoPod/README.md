# Starflower Eco Pod ★

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/objects/farmables/alta/ground/stardust/pod/icon.png" alt="Starflower Eco Pod ★ icon" loading="lazy" width="auto" height="16px"/> **Starflower Eco Pod ★** is an essential plantable item.

Contains a modified stardust sapling, as well as all things necessary for it to ripe. Can be placed anywhere.
