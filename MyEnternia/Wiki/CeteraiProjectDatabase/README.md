# Ceterai Project Database

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/codex/alta/datamass/plasma.png" alt="Ceterai Project Database icon" loading="lazy" height=16px width="auto" /> **Ceterai Project Database** is a legendary codex entry.

A list of unknown devices. It seems like it's missing some entries. Code on the back reads 'CEN-**0'.  
This is a datamass containing information about the Ceterai Project, a research and development initiative focused on advanced technology and energy systems. The entries are organized by their registration date and connections to other entries, with each entry assigned a unique identifier in the CTN format (thr. CEN-202).

## Ingame

Species descriptions:

- Alta: Oh no... I hope this isn't a bad sign.

Radiomessages on pickup:

- You've found an alta datamass. These tiny portable databases tend to contain important encrypted info used by alta crafting stations to unlock various blueprints.  
These can either be crafted or found in random alta pods on alterash planets.
