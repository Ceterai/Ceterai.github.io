# Onitti Motsu Soup

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/items/generic/food/tier3/ct_onitti_motsu.png" alt="Onitti Motsu Soup icon" loading="lazy" width="auto" height="16px"/> **Onitti Motsu Soup** is a rare dish.

A very nutritious herb soup.  
One of the classic motsu recipes that captures the essence of the concept quite well. The simplicity of the recipe is a big advantage too.
