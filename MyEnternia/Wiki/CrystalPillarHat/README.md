# Crystal Pillar Hat

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/items/armors/misc/ct_misc_crystal_pillar_hat/icon.png" alt="Crystal Pillar Hat icon" loading="lazy" width="auto" height="16px"/> **Crystal Pillar Hat** is a rare head cosmetic item.

A bright blue crystal that you can put on your head.

## Usage

### Dyeing

Can be dyed with any clothing <img src="https://starbounder.org/mediawiki/images/c/cf/Dye_Remover.png" alt="Dye icon" width="8" height="12"/> [Dyes](https://starbounder.org/Dye), including [custom dyes](https://ceterai.github.io/MyEnternia/Wiki/Dyes) from this mod.
