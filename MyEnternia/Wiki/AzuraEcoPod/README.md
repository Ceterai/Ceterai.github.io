# Azura Eco Pod ★

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/objects/farmables/alta/ground/azura/pod/icon.png" alt="Azura Eco Pod ★ icon" loading="lazy" width="auto" height="16px"/> **Azura Eco Pod ★** is a legendary plantable item.

Contains a modified striped wood sapling, as well as all things necessary for it to ripe. Can be placed anywhere.
