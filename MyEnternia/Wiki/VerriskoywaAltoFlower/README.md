# Verriskoywa Alto Flower

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/objects/biome/alterash/koywa/flowers/alto/icon.png" alt="Verriskoywa Alto Flower icon" loading="lazy" width="auto" height="16px"/> **Verriskoywa Alto Flower** is a common breakable object.

Tall, blue reed flower.  
It stands out among seemingly endless vasts of koywa reeds.

## Ingame

- Floran: Reedsss.
- Glitch: Disinterested. Just reeds.
- Human: Reeds.
- Hylotl: Tall, leafy reeds.
- Novakid: It's just some reeds.

</details>

This object **doesn't drop itself** when broken.
