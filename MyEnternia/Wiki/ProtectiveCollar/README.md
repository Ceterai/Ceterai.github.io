# Protective Collar

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/items/armors/alta/other/protective_collar/icon.png" alt="Protective Collar icon" loading="lazy" width="auto" height="16px"/> **Protective Collar** is an uncommon head cosmetic item.

This collar provides an energy shield to prevent you from receiving damage.

## Parameters

Applies effects:

### Crafting

Can be used to craft:

- <img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/items/armors/alta/tier4/security/pack/icon.png" alt="Life Support Pack icon" loading="lazy" width="auto" height="16px"/> [Life Support Pack](https://ceterai.github.io/MyEnternia/Wiki/LifeSupportPack)

### Dyeing
