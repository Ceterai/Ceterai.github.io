# Alta Tall Flower Pot

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/objects/alta/special/tools/pots/tall/icon.png" alt="Alta Tall Flower Pot icon" loading="lazy" width="auto" height="16px"/> **Alta Tall Flower Pot** is a common storage unit.

A simple laminated wooden flower pot on a tall extension for flowers that like more soil.  
Can be used for capturing plant critters for collection and to have them as pets.
