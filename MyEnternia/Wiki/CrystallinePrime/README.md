# Crystalline Prime

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/items/active/alta/loot/biome/ct_crystalline_prime_loot.png" alt="Crystalline Prime icon" loading="lazy" width="auto" height="16px"> **Crystalline Prime** is a planetary [biome](https://ceterai.github.io/MyEnternia/Wiki/Biomes).

Also known as Calline Prime, this is an <img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/interface/bookmarks/icons/ct_alterash_prime_planet.png" alt="icon" loading="lazy" width="auto" height="16px"> [alterash prime](https://ceterai.github.io/MyEnternia/Wiki/AlterashPrime) variation of the original <img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/items/active/alta/loot/biome/ct_crystalline_prime_loot.png" alt="Crystalline icon" loading="lazy" width="16px" height="16px"> [crystalline](https://starbounder.org/Crystalline) biome, caused by what's called a <img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/codex/alta/ebook/crystalline_prime.png" alt="Crystalline Primification icon" loading="lazy" width="auto" height="16px"> [crystalline primification](https://ceterai.github.io/MyEnternia/Wiki/CrystallinePrimification).  
These [crystallic](https://ceterai.github.io/MyEnternia/Wiki/Io#crystallic) caves are dimly lit via soft green light emitted from the <img src="https://starbounder.org/mediawiki/images/3/31/Crystal.png" alt="Crystal icon" loading="lazy" width="12px" height="16px"> [crystals](https://starbounder.org/Crystal) that grow here. Some of those crystals are edible, by the way.
