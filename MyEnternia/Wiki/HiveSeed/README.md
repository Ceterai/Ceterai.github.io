# Hive Seed

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/objects/farmables/alta/liquid/hive/icon.png" alt="Hive Seed icon" loading="lazy" width="auto" height="16px"/> **Hive Seed** is an uncommon plantable item.

A small growing hive full of fresh edible bugs. Requires to be at least 50% in a liquid.

## Ingame

Species descriptions:

- Alta: I wonder if this larva plant has a scientific name...

Radiomessages on pickup:

- I see that you've discovered some seeds. I suggest planting them in some tilled soil, and watering them until they grow. The tools you need can be built at a foraging table.

2. Lasts from 570 to 630 seconds.
