# Yaavikada

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/assetMissing.png" alt="Yaavikada icon" loading="lazy" height="16px" width="auto" /> **Yaavikada** is an uncommon dish.

Even though is looks like any other _alta kada_, this is actualy a full-on baked assorti in a <img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/assetMissing.png" alt="Yaara Melon icon" loading="lazy" height="16px" width="auto" /> [yaara melon](https://ceterai.github.io/MyEnternia/Wiki/YaaraMelon) skin.  
As something in between a kada and an assorti, this is similarly difficult to prepare and would require a chef's hand to make it work.
