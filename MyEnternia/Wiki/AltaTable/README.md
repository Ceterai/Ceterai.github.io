# Alta Table

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/objects/alta/basic/table/icon.png" alt="Alta Table icon" loading="lazy" height="16px" width="auto" /> **Alta Table** is a common decorative object.

A simplistic table made out of titanium. Only 6 blocks wide.  
Most basic alta furniture is designed to be simple, lightweight, and, most importantly, easy to clean and disassemble. The goal is to be comfortable and portable at the same time, while also being easy to recycle if needed.
