# Verriskoywa Sapling

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/objects/farmables/alta/liquid/koywa/sapling/icon.png" alt="Verriskoywa Sapling icon" loading="lazy" height="16px" width="auto" /> **Verriskoywa Sapling** is an uncommon plantable item.

A proper sapling that takes less space and has increased and more consistent speed of growth stages. Requires to be at least 50% in a liquid.

## Ingame

Species descriptions:

- Alta: Since koywa grass is so widely used, it only makes sense to make it farmable~
