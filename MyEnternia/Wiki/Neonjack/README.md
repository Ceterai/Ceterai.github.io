# Neonjack

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/objects/alta/special/events/ceternity/neonjack/icon.png" alt="Neonjack icon" loading="lazy" width="auto" height="16px"> **Neonjack** is an uncommon light source.

A "Neonjack", as altas call them. A festive neonmelon decoration with Ceternity's face carved into it that glows eerily in the dark.  
A neonmelon carved into a jack o' lantern, complete with an internal light source. It flickers eerily in the dark, making it perfect for Halloween decorations.
