# Eco Ela Sapling

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/objects/farmables/alta/liquid/ela/eco/icon.png" alt="Eco Ela Sapling icon" loading="lazy" height="16px" width="auto" /> **Eco Ela Sapling** is a rare plantable item.

This glow fibre sapling was genetically modified to fit into small spaces and consume almost no liquids.

## Ingame

Species descriptions:
