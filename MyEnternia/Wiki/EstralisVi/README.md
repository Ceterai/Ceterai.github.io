# Estralis Vi

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/items/active/weapons/ranged/alta/wrist/ct_estralis_vi.png" alt="Estralis Vi icon" loading="lazy" width="auto" height="16px"> **Estralis Vi** is a rare one-handed electric unique weapon.

A unique wrist device lost in the midst of nowhere. How did it get here? Is it... looking at you?

Can be upgraded to <img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/items/active/weapons/ranged/alta/wrist/ct_estralis_vi_2.png" alt="Estria's Vyurm ★ icon" loading="lazy" width="auto" height="16px"> [Estria's Vyurm ★](https://ceterai.github.io/MyEnternia/Wiki/Estria'sVyurm) at the Weapon Upgrade Anvil.

## Ingame

Radiomessages on pickup:

## Parameters

### Plasmastrains

The core of this blaster is able to generate Plasmic Strains able to change their form.

Parameters:

- Base Dps: 4
- Chain:
  - Taper: 0.5
  - Segment Image: `/items/active/weapons/ranged/alta/abils/tentacles/ct_estralis_vi_s.png`
  - End Segment Image: `/items/active/weapons/ranged/alta/abils/tentacles/ct_estralis_vi_e.png`
  - Segment Size: 1.0
- Energy Usage: 16
- Fire Time: 0.2
- Guide Projectiles: `True`

- Max Length: 20

- Projectile Parameters:
