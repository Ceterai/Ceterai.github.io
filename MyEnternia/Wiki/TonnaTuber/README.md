# Tonna Tuber

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/objects/farmables/alta/liquid/tonna/icon.png" alt="Tonna Tuber icon" loading="lazy" width="auto" height="16px"/> **Tonna Tuber** is an uncommon plantable item.

An alta-engineered small plant that grows tonna nuts at a smaller scale. Requires to be at least 50% in a liquid.

## Ingame

Species descriptions:

- Alta: Now I don't have to climb a tonnova to get me some tonnas!

Radiomessages on pickup:

- I see that you've discovered some seeds. I suggest planting them in some tilled soil, and watering them until they grow. The tools you need can be built at a foraging table.

## Parameters

Growing stages:

1. Lasts from 570 to 630 seconds.
2. Lasts from 570 to 630 seconds.
3. Lasts from 1670 to 1730 seconds.
