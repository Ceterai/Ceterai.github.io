# Alta Gateway

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/objects/alta/city/gateway/icon.png" alt="Alta Gateway icon" loading="lazy" width="auto" height="16px"/> **Alta Gateway** is a common door.

This door is airtight and made with incredibly strong materials.

## Ingame

<details markdown="1"><summary>Species descriptions: (Expand 8 items)</summary>

- Alta: A regular blast door, used primarily in alta cities.

- Avian: A door that opens at a curious angle.
- Floran: This door looks heavy. Floran doesn't want to get sssquished.
- Glitch: Intrigued. This doorway appears to be leaning.
