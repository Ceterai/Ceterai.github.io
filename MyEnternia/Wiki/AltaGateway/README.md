# Alta Gateway

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/objects/alta/city/gateway/icon.png" alt="Alta Gateway icon" loading="lazy" height="16px" width="auto" /> **Alta Gateway** is a common door.

This door is airtight and made with incredibly strong materials.

## Ingame
