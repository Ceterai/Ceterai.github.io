# Yaara Trousers

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/items/armors/alta/tier6/ceterai/legwear/icon.png" alt="Yaara Trousers icon" loading="lazy" width="auto" height="16px"/> **Yaara Trousers** is an uncommon leg cosmetic item.

These trousers make a pleasant woody sound when walking.  
Worth noting that the wooden texture of this peace of costume doesn't reflect it's protective capabilities.

## Ingame

Species descriptions:

- Alta: Very... planty?
- Floran: Floran dissquissse ass plant. Ambush prey. Win fightss.

## Usage

### Dyeing

Can be dyed with any clothing <img src="https://starbounder.org/mediawiki/images/c/cf/Dye_Remover.png" alt="Dye icon" width="8" height="12"/> [Dyes](https://starbounder.org/Dye), including [custom dyes](https://ceterai.github.io/MyEnternia/Wiki/Dyes) from this mod.
