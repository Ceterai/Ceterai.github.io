# Alta Snow Cannon

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/items/active/alta/toys/snow/cannon/ct_alta_snow_cannon.png" alt="Alta Snow Cannon icon" loading="lazy" width="auto" height="16px"> **Alta Snow Cannon** is an uncommon two-handed cryogenic toy.

Powered by snow, it's fully automatic and keeps your gloves dry.

## Parameters

### Primary Ability

Main ability of the item, activated with left mouse button.

Parameters:

- Base Dps:
- Energy Usage: 24
- Fire Time: 0.35
- Fire Type: `auto`
- Inaccuracy: 0.02
- Projectile Count: 1
- Projectile Parameters:
  - Speed: 60
  - Knockback: 10
- Projectile Type: `ct_snow_block`
