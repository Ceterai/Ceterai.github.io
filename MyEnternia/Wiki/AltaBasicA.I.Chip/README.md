# Alta Basic A.I. Chip

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/items/aichips/ct_alta_ai_chip.png" alt="Alta Basic A.I. Chip icon" loading="lazy" width="auto" height="16px"/> **Alta Basic A.I. Chip** is an uncommon AI chip.

An alta ship A.I. interface chip. _With Customizable A.I.: Can be installed on S.A.I.L.'s terminal. Visuals only._  
This A.I. chip is constructed by interplanetary standards, though its software is purely alta-native, meaning it performs similar to a datamass.

## Ingame

Species descriptions:

- Alta: A simple interface in a highly-compatible datamass. Neat.
