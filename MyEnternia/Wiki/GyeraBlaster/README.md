# Gyera Blaster

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/items/active/alta/tools/other/ct_gyera_blaster.png" alt="Gyera Blaster icon" loading="lazy" width="auto" height="16px"/> **Gyera Blaster** is a legendary one-handed tool.

This energy tool can spread different omni grass seeds to fill any room with green covers!

## Ingame

Radiomessages on pickup:

- You've just picked up an alta weapon, or an "energy tool", as they call it. According to my database, these usually have an extended set of features. I suggest reading its description.

## Parameters

### Alterash Gardening

Press to disperse seeds! Hold to change seed type. Available: regular grass, flowery, lush and thick.

Parameters:

- Class: `AltaBlasterSwitch`
- Base Dps: 4
- Energy Usage: 16
- Fire Time: 0.3
- Default Fire Type: `grass`
- Fire Types:
  - Grass:
    - Type: `blast`
    - Params:
      - Type: `ct_omni_grass_spray`
    - Hold Time Max: 0.75
  - Flowery:
    - Type: `blast`
    - Params:
      - Type: `ct_omni_flowery_grass_spray`
    - Hold Time Max: 0.75
  - Lush:
    - Type: `blast`
    - Params:
      - Type: `ct_omni_lush_grass_spray`
    - Hold Time Max: 0.75
  - Thick:
    - Type: `blast`
    - Params:
      - Type: `ct_omni_thick_grass_spray`
    - Hold Time Max: 0.75
