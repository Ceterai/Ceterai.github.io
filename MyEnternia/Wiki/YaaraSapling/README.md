# Yaara Sapling

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/objects/farmables/alta/liquid/yaara/sapling/icon.png" alt="Yaara Sapling icon" loading="lazy" width="auto" height="16px"/> **Yaara Sapling** is an uncommon plantable item.

A proper sapling that takes less space and has increased and more consistent speed of growth stages.

## Ingame

Species descriptions:

- Alta: A yaara sprout able to live on its own is a pretty rare occurrence!

2. Lasts from 380 to 420 seconds.
