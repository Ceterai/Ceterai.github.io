# Nia Cuisine Book

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/codex/alta/ebook/gyera.png" alt="Nia Cuisine Book icon" loading="lazy" height="16px" width="auto" /> **Nia Cuisine Book** is an uncommon codex entry.

An ebook containing various energy-based [alta](https://ceterai.github.io/MyEnternia/Wiki/Tags/Alta) recipes and their history. (Unlocks some nia recipes)  
This [ebook](https://ceterai.github.io/MyEnternia/Wiki/Tags/Ebook) describes different nia recipes and the culture around them.

## Ingame

Species descriptions:

- Alta: Energy-based bood is the best, considering our bodies run on plasma...

Radiomessages on pickup:

- You've found an alta ebook - a collimator electronic reader usually used as a reusable substitute for books. Often crafted in alta datacenters.
