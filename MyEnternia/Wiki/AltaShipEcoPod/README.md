# Alta Ship Eco Pod

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/objects/alta/special/tools/pods/ship/icon.png" alt="Alta Ship Eco Pod icon" loading="lazy" width="auto" height="16px"/> **Alta Ship Eco Pod** is a legendary tree sapling.

An empty enterite chamber, presumably made for plants.  
The capabilities are rather limited, yet even still this is a fascinating peace of technology able to keep a growing lifeform alive and well.
