# Unwarped Fly Egg

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/items/active/alta/spawners/bugs/ct_unwarped_fly_egg.png" alt="Unwarped Fly Egg icon" loading="lazy" height="16px" width="auto" /> **Unwarped Fly Egg** is an uncommon throwable item.

Can be used to spawn a bug called <img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/objects/alta/special/bugs/unwarped_fly/icon.png" alt="Unwarped Fly icon" loading="lazy" height="16px" width="auto" /> [unwarped fly](https://ceterai.github.io/MyEnternia/Wiki/UnwarpedFly), a much friendlier version of a [warped fly](https://ceterai.github.io/MyEnternia/Wiki/warpedfly).  
These "eggs" are really just highly incubated versions of creatures, used by altas during research, often for simulation purposes, which is why they tend to be friendly to their owner.
