# Haven Eco Pod ★★

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/objects/farmables/alta/ground/haven/pod/icon.png" alt="Haven Eco Pod ★★ icon" loading="lazy" width="auto" height="16px"/> **Haven Eco Pod ★★** is a legendary plantable item.

Contains a modified petal sapling, as well as all things necessary for it to ripe. Can be placed anywhere.
