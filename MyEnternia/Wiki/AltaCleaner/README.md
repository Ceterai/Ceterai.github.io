# Alta Cleaner

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/items/active/alta/tools/brushes/cleaner/icon.png" alt="Alta Cleaner icon" loading="lazy" width="auto" height="16px"> **Alta Cleaner** is an uncommon tool.
