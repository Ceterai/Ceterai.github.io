# Yaara Crystal Flower

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/objects/biome/alterash/yaara/decorative/lamp/ct_yaara_lamp_crystal_icon.png" alt="Yaara Crystal Flower icon" loading="lazy" width="auto" height="16px"/> **Yaara Crystal Flower** is a common light source.

A not-sunflower lamp covered in crystals. Looks like they grow right from the plant.
