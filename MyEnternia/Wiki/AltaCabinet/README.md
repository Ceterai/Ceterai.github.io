# Alta Cabinet

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/objects/alta/basic/cabinet/icon.png" alt="Alta Cabinet icon" loading="lazy" height="16px" width="auto" /> **Alta Cabinet** is a common storage unit.

A simple titanium locker. Has mechanical connectors to stack these on top of each other. **Is a platform. Can be placed on the ground, walls and ceiling.**  
Most basic alta furniture is designed to be simple, lightweight, and, most importantly, easy to clean and disassemble. The goal is to be comfortable and portable at the same time, while also being easy to recycle if needed.
