# Iced Tsay Dessert ★

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/items/generic/food/tier2/ct_tsay_icecream.png" alt="Iced Tsay Dessert ★ icon" loading="lazy" height="16px" width="auto" /> **Iced Tsay Dessert ★** is an uncommon dish.

This icy creamy dessert is sparkling with sourness! Can you lick it without flinching?  
Almost like a lemon ice cream, this dessert can be decieving in its presentation.
