# Boosted Ionic Sapling

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/objects/farmables/alta/liquid/ionic/boosted/icon.png" alt="Boosted Ionic Sapling icon" loading="lazy" height="16px" width="auto" /> **Boosted Ionic Sapling** is a legendary plantable item.

This sapling comes with enhanced mutations that allow much faster ripening. Requires to be at least 50% in a liquid.

## Ingame

Species descriptions:
