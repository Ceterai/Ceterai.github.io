# Warped Hive Sapling

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/objects/farmables/alta/ground/warped_hive/sapling/icon.png" alt="Warped Hive Sapling icon" loading="lazy" width="auto" height="16px"/> **Warped Hive Sapling** is an uncommon plantable item.

A proper sapling that takes less space and has increased and more consistent speed of growth stages.
