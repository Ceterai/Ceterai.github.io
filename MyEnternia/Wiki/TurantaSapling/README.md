# Turanta Sapling

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/objects/farmables/alta/ground/turanta/sapling/icon.png" alt="Turanta Sapling icon" loading="lazy" height="16px" width="auto" /> **Turanta Sapling** is an uncommon plantable item.

A proper sapling that takes less space and has increased and more consistent speed of growth stages.
