# Omni Eco Pod

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/objects/alta/special/tools/pods/omni/icon.png" alt="Omni Eco Pod icon" loading="lazy" width="auto" height="16px"/> **Omni Eco Pod** is an essential tree sapling.

An empty celestial chamber, presumably made for plants.  
The capabilities are rather limited, yet even still this is a fascinating peace of technology able to keep a growing lifeform alive and well.
