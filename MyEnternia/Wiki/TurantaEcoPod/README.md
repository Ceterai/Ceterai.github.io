# Turanta Eco Pod

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/objects/farmables/alta/ground/turanta/pod/icon.png" alt="Turanta Eco Pod icon" loading="lazy" width="auto" height="16px"/> **Turanta Eco Pod** is a legendary plantable item.

Contains a modified giant petal sapling, as well as all things necessary for it to ripe. Can be placed anywhere.
