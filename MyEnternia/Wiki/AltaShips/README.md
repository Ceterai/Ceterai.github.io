# Alta Ships

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/codex/alta/ebook/ship.png" alt="Alta Ships icon" loading="lazy" height="16px" width="auto" /> **Alta Ships** is a legendary codex entry.

A piece of information about alta spaceports and space travel industry.  
This document explains the common design choices and safety protocols behind most things you can find on an alta ship.

## Ingame

Species descriptions:

- Alta: Safety is important, so I'm glad that decisions around modular ship parts are being given this much thought.

Radiomessages on pickup:

- You've found an alta ebook - a collimator electronic reader usually used as a reusable substitute for books. Often crafted in alta datacenters.
