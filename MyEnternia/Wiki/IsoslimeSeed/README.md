# Isoslime Seed

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/objects/farmables/alta/ground/isoslime/icon.png" alt="Isoslime Seed icon" loading="lazy" height=16px width="auto" /> **Isoslime Seed** is a common plantable item.

A plant covered in slime. It's a bit cold to the touch.

## Ingame

Species descriptions:

- Alta: This slimy plant can produce small amounts of izopoi!

Radiomessages on pickup:

- I see that you've discovered some seeds. I suggest planting them in some tilled soil, and watering them until they grow. The tools you need can be built at a foraging table.
