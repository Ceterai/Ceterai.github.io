# Alta Lab Eco Pod

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/objects/alta/special/tools/pods/lab/icon.png" alt="Alta Lab Eco Pod icon" loading="lazy" width="auto" height="16px"/> **Alta Lab Eco Pod** is a rare tree sapling.

An empty bion chamber, presumably made for plants.  
The capabilities are rather limited, yet even still this is a fascinating peace of technology able to keep a growing lifeform alive and well.
