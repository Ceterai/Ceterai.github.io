# Alta Bed

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/objects/alta/basic/bed/icon.png" alt="Alta Bed icon" loading="lazy" height=16px width="auto" /> **Alta Bed** is a common piece of furniture.

A simplistic titanium bed. Not too comfy. Has a compartment with a built-in night light.  
Most basic alta furniture is designed to be simple, lightweight, and, most importantly, easy to clean and disassemble. The goal is to be comfortable and portable at the same time, while also being easy to recycle if needed.
