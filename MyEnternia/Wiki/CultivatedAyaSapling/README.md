# Cultivated Aya Sapling

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/objects/farmables/alta/main/aya/cultivated/icon.png" alt="Cultivated Aya Sapling icon" loading="lazy" height=16px width="auto" /> **Cultivated Aya Sapling** is an uncommon plantable item.

Cultivation further improves speed and stability of growth, but also of ripening.

## Ingame

Species descriptions:

- Alta: A lab-produced aya spring. Seems like this one won't grow into a tall ayaka, like usual.
