# Boosted Vionora Sapling

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/objects/farmables/alta/liquid/viona/boosted/icon.png" alt="Boosted Vionora Sapling icon" loading="lazy" width="auto" height="16px"/> **Boosted Vionora Sapling** is a legendary plantable item.

This sapling comes with enhanced mutations that allow much faster ripening. Requires to be at least 50% in a liquid.

## Ingame

Species descriptions:
