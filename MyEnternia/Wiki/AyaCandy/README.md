# Aya Candy

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/items/generic/food/tier2/ct_aya_candy.png" alt="Aya Candy icon" loading="lazy" height="16px" width="auto" /> **Aya Candy** is an uncommon dish.

Made with aya jam, it's a sour and sweet lollipop!  
Usually prepared and sold in packs, these are great to share with friends and are perfect if you really need to not think of anything.
