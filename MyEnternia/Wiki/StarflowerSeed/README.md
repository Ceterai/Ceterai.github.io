# Starflower Seed

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/objects/farmables/alta/ground/stardust/icon.png" alt="Starflower Seed icon" loading="lazy" height="16px" width="auto" /> **Starflower Seed** is a legendary plantable item.

A flower-like plant with colorful, glittery specks all over it.

## Ingame

Species descriptions:

- Alta: A stardust plant?! They exist?!

Radiomessages on pickup:

- I see that you've discovered some seeds. I suggest planting them in some tilled soil, and watering them until they grow. The tools you need can be built at a foraging table.
