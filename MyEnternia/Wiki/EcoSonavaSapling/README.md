# Eco Sonava Sapling ★

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/objects/farmables/alta/ground/sona/eco/icon.png" alt="Eco Sonava Sapling ★ icon" loading="lazy" width="auto" height="16px"> **Eco Sonava Sapling ★** is an uncommon plantable item.

This snow fibre sapling was genetically modified to fit into small spaces and consume almost no liquids.
