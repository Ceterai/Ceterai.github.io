# Combat Protocol Datamass

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/codex/alta/datamass/combat.png" alt="Combat Protocol Datamass icon" loading="lazy" height=16px width="auto" /> **Combat Protocol Datamass** is a rare codex entry.

A list of short descriptions of some kind of regulations. Parts of data are corrupted. Code on the back reads 'CEN-411'.  
The regulations include regular alta combat protocols, special operations guidelines, and weaponized conflict strategies. The document is partially corrupted, with some entries missing or unreadable. The code on the back indicates that this is part of a larger set of documents related to combat protocols.

## Ingame

Species descriptions:

- Alta: I hope I never have to use this...

Radiomessages on pickup:

- You've found an alta datamass. These tiny portable databases tend to contain important encrypted info used by alta crafting stations to unlock various blueprints.  
These can either be crafted or found in random alta pods on alterash planets.
