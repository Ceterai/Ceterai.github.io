# Ion Nade

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/items/throwables/ct_ionic_nade.png" alt="Ion Nade icon" loading="lazy" height="16px" width="auto" /> **Ion Nade** is a rare electric throwable item.

A grenade that creates a localised static field that stuns and damages enemies.
