# Alta Bucket

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/items/active/alta/tools/water/bucket/icon.png" alt="Alta Bucket icon" loading="lazy" height=16px width="auto" /> **Alta Bucket** is a common two-handed tool.

A simple ayaka bucket to water your crops in bulk. Make sure you have a drain.
