# Alta Lab Tanks

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/objects/alta/lab/tanks/icon.png" alt="Alta Lab Tanks icon" loading="lazy" height=16px width="auto" /> **Alta Lab Tanks** is an uncommon decorative object.

Empty interconnected tanks. The glass seems to be made out of crystals.
