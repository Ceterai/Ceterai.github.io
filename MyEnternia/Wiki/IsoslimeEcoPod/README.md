# Isoslime Eco Pod

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/objects/farmables/alta/ground/isoslime/pod/icon.png" alt="Isoslime Eco Pod icon" loading="lazy" width="auto" height="16px"> **Isoslime Eco Pod** is a legendary plantable item.

Contains a modified isoslime sapling, as well as all things necessary for it to ripe. Can be placed anywhere.
