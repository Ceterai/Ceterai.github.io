# Poi Eco Pod

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/objects/farmables/alta/ground/isoslime/pod_green/icon.png" alt="Poi Eco Pod icon" loading="lazy" width="auto" height="16px"> **Poi Eco Pod** is a legendary plantable item.

Contains a modified slime sapling, as well as all things necessary for it to ripe. Can be placed anywhere.
