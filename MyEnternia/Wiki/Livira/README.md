# Livira

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/objects/alta/special/plants/trees/ct_livira_tree.png" alt="Livira icon" loading="lazy" width="auto" height="16px"> **Livira** is a rare tree sapling.

A very rare alien tree with unusual circular formations at the top.  
This tree consists of a special type of wood often used in decorative furniture crafting.
