# Neonmelon Sapling

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/objects/farmables/alta/ground/produce/neonmelon/sapling/icon.png" alt="Neonmelon Sapling icon" loading="lazy" width="auto" height="16px"/> **Neonmelon Sapling** is an uncommon plantable item.

A proper sapling that takes less space and has increased and more consistent speed of growth stages.

## Ingame

Species descriptions:

- Alta: I thought only yaara melons could get this big...

## Parameters

Growing stages:

1. Lasts from 760 to 840 seconds.
2. Lasts from 1600 to 1800 seconds.
3. Can be harvested:
`neonmelonHarvest`
