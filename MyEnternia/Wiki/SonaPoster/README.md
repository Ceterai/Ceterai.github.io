# Sona Poster

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/objects/alta/special/events/sonaveil/ct_sona_poster.png" alt="Sona Poster icon" loading="lazy" width="auto" height="16px"> **Sona Poster** is a common decorative object.

A poster depicting a snowalta.  
This festive poster features a snowalta - an alta-shaped snowman figure that has become the beloved symbol of Sona's Veil celebrations. Snowaltas are built during the cold solstice season as a joyful winter activity, using snow, crystals, ayaka branches and fruit noses. They are often decorated with scarves, hats, and string lights to add warmth and cheer.
