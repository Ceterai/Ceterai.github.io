# Turanta Seed

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/objects/farmables/alta/ground/turanta/icon.png" alt="Turanta Seed icon" loading="lazy" height="16px" width="auto" /> **Turanta Seed** is an uncommon plantable item.

A lush flower with big petals that is constantly growing.

## Ingame

Species descriptions:

- Alta: A pretty turanta! Its big petals can be quite useful!

Radiomessages on pickup:

- I see that you've discovered some seeds. I suggest planting them in some tilled soil, and watering them until they grow. The tools you need can be built at a foraging table.
