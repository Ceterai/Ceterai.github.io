# Ghearun M1 Forcefield

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/items/armors/alta/tier6/ghearun/back/icon.png" alt="Ghearun M1 Forcefield icon" loading="lazy" height=16px width="auto" /> **Ghearun M1 Forcefield** is a legendary epp item.

Automatically applies a rechargable shield. Provides enviro protection. Has an augment slot. Runs on [gheatsyn](https://ceterai.github.io/MyEnternia/Wiki/Tags/Gheatsyn).

## Ingame

Radiomessages on pickup:

- You've got a hold of a level 6+ alta back armor piece. Aside from providing special effects, these usually have an augment slot, allowing you to enhance them with an augment of your choice.
