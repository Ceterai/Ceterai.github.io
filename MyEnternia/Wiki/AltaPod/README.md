# Alta Pod

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/objects/alta/city/pod/icon.png" alt="Alta Pod icon" loading="lazy" height="16px" width="auto" /> **Alta Pod** is an uncommon storage unit.

A simplistic pod that can be used as a chest.  
Portable alta containers, or alta pods for short, are a standardized way of transporting equipment in a safe, airtight environment.
