# Alta Hoe

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/items/active/alta/tools/till/hoe/icon.png" alt="Alta Hoe icon" loading="lazy" height="16px" width="auto" /> **Alta Hoe** is a common one-handed tool.

A simple tool to till soil. Not very accurate. The tilled soil can then be used to grow all sorts of crops!
