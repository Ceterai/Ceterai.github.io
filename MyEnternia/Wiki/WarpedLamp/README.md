# Warped Lamp

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/objects/biome/alterash/warped/decorative/lamp/icon.png" alt="Warped Lamp icon" loading="lazy" width="auto" height="16px"/> **Warped Lamp** is a common light source.

Some red vines with glowing bits entangled into a lamp-like form. Are they alive?
