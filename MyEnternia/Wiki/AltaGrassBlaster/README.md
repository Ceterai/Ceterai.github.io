# Alta Grass Blaster ★★

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/items/active/alta/tools/other/ct_alta_grass_blaster.png" alt="Alta Grass Blaster ★★ icon" loading="lazy" height=16px width="auto" /> **Alta Grass Blaster ★★** is a rare one-handed tool.

This energy tool can spread regular omni grass seeds to fill any room with green covers!
