# Hive Curry

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/items/generic/food/tier3/ct_hive_curry.png" alt="Hive Curry icon" loading="lazy" height="16px" width="auto" /> **Hive Curry** is a rare dish.

Healthy, spicy, squishy. Do you like bugs?  
The larva themself are considered a very rarily used ingredient, since altas are vegetarian by nature, but this dish utilizes them in moderate amounts.
