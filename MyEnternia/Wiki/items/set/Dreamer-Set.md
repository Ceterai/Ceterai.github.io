![ ](https://raw.githubusercontent.com/Ceterai/Enternia/main/items/active/alta/sets/dreamer.png) **Dreamer Set** is a legendary lvl.6 set.  
Can be sold for *2240* <img src="https://starbounder.org/mediawiki/images/2/21/Pixel.png" width="12" height="16"/> [pixels](https://starbounder.org/Pixel).  
Contains full equipment set of an [[ alta|Alta ]] dreamer, including a lore codex of this set.
Is cheaper to craft this way than separately.  
A set that emphasizes a dreamer's surge of enternia - it supplies you with life energy, making it safer to deal with this much enternia, but also uses it against you targets, making it easier to deal with more aggressive prey.
