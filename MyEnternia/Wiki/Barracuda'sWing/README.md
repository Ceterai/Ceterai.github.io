# Barracuda's Wing ★

<img src="https://raw.githubusercontent.com/wiki/Ceterai/Enternia/images/icons/ct_tool_mimic-horn-upgrade.png" alt="Barracuda's Wing ★ icon" loading="lazy" width="auto" height="16px"> **Barracuda's Wing ★** is a legendary two-handed poisonous broadsword.

A horn from an especially massive cuda lurking in the flooded alterash caves.  
It looks polished due to special cover that can be found on some appendages of rare cudas.

An upgrade for <img src="https://raw.githubusercontent.com/wiki/Ceterai/Enternia/images/icons/ct_tool_mimic-horn.png" alt="Barracuda's Horn icon" loading="lazy" width="auto" height="16px"> [Barracuda's Horn](https://ceterai.github.io/MyEnternia/Wiki/Barracuda'sHorn).

## Parameters

### Primary Ability

Main ability of the item, activated with left mouse button.

Parameters:

- Base Dps: 13
- Fire Time: 1.0

### Secondary Ability

Alternative ability of the item, activated with right mouse button.

Parameters:

- Energy Usage: 80
- Projectile Type: `webblast`
- Projectile Offset:  4.5,  0.5
- Base Damage: 1
