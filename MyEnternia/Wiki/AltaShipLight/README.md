# Alta Ship Light

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/objects/alta/ship/light/icon.png" alt="Alta Ship Light icon" loading="lazy" width="auto" height="16px"/> **Alta Ship Light** is a rare light source.

A tiny wall light made with isolating airtight materials. Works perfectly even in outer space.
