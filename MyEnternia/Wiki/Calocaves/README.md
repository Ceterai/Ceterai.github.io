# Calocaves

Calocaves, or [Calin](https://ceterai.github.io/MyEnternia/Wiki/Tags/Calin) for short, are caves with rich deposits of plain regular [crystals](https://ceterai.github.io/MyEnternia/Wiki/Tags/Crystal).
