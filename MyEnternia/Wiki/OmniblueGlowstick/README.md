# Omniblue Glowstick

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/items/active/alta/glowsticks/omniblue.png" alt="Omniblue Glowstick icon" loading="lazy" height=16px width="auto" /> **Omniblue Glowstick** is an uncommon throwable item.

Creates an omniblue bioluminescent glow when thrown. Is made from rare crystallic materials.  
Often used by [arknights](https://ceterai.github.io/MyEnternia/Wiki/Arknight) and other [eva](https://ceterai.github.io/MyEnternia/Wiki/Tags/Eva) altas for aircraft navigation. Different colors mean different responses, and same with movements.

## Ingame

Species descriptions:

- Alta: The color of altas! I've heard that this is also Ceterai's favourite color too...
