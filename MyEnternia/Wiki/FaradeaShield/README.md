# Faradea Shield

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/items/active/shields/ct_eds_shield.png" alt="Faradea Shield icon" loading="lazy" height="16px" width="auto" /> **Faradea Shield** is a rare shield.

An EDS shield repurposed by Faradea mercs.  
Gorgeous blue and yellow are symbolizing freedom, the goal and desire of every Faradea merc.

## Ingame

Species descriptions:

- Alta: Those blue and yellow look beautiful on this shield!

Radiomessages on pickup:

- You're holding an alta shield. My database shows an enhanced set of features, loading a proper description for you. Protection is known to be a prominent topic in alta culture and a big research field as well, so this might be worth checking out.
- Seems like the item you just picked up belongs to Faradea - an independent alta mercenary faction. Let's hope you don't find the previous owner of this item.
