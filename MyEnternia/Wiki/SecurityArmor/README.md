# Security Armor

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/items/armors/alta/tier4/security/chest/icon.png" alt="Security Armor icon" loading="lazy" width="auto" height="16px"/> **Security Armor** is an uncommon chest armor piece.

A standard protocol armor piece for [alta](https://ceterai.github.io/MyEnternia/Wiki/Tags/Alta) security units.

Can be upgraded to <img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/items/armors/alta/tier4/security/chest/icon.png" alt="GR Guard ★ icon" loading="lazy" width="auto" height="16px"/> [GR Guard ★](https://ceterai.github.io/MyEnternia/Wiki/GRGuard) at the Weapon Upgrade Anvil.
