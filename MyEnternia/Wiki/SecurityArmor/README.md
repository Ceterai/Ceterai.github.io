# Security Armor

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/items/armors/alta/tier4/security/chest/icon.png" alt="Security Armor icon" loading="lazy" height=16px width="auto" /> **Security Armor** is an uncommon chest item.

A standard protocol armor piece for [alta](https://ceterai.github.io/MyEnternia/Wiki/Tags/Alta) security units.
