# Alta Security A.I. Chip

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/items/aichips/ct_alta_security_ai_chip.png" alt="Alta Security A.I. Chip icon" loading="lazy" height=16px width="auto" /> **Alta Security A.I. Chip** is an uncommon AI chip.

An alta security comms chip. _With Customizable A.I.: Can be installed on S.A.I.L.'s terminal._  
This A.I. chip is constructed by interplanetary standards, though its software is purely alta-native, meaning it performs similar to a datamass.

## Ingame

Species descriptions:

- Alta: A security interface in a highly-compatible datamass.
