# Sonaveil Scarf

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/items/armors/alta/events/sonaveil/scarf/icon.png" alt="Sonaveil Scarf icon" loading="lazy" width="auto" height="16px"> **Sonaveil Scarf** is a legendary head cosmetic item.

Better wrap up - wouldn't want you catching a cold out there.  
This is a cozy holiday scarf, hand-knitted by an alta. Soft, warm, and carrying that familiar holiday charm. During Sonaveil, altas exchange hand-knitted scarves like this one as tokens of warmth and friendship.

## Ingame

Species descriptions:

- Alta: A finely crafted scarf, perfect for the cold solstice! Wish I could knit like this so I could make a gift for my sis...

### Dyeing
