# Sonaveil Gift

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/items/active/alta/loot/events/sonaveil.png" alt="Sonaveil Gift icon" loading="lazy" width="auto" height="16px"/> **Sonaveil Gift** is a legendary loot item.

Contains some perfect Sonaveil gifts wrapped in true holiday spirit!  
This small loot crate is filled with various treats and goodies and is meant to be shared with your sisters around the cold solstice.

## Ingame

Species descriptions:

- Alta: A hope there's a warm knitted scarf inside!

Radiomessages on pickup:

- I detected an alta loot crate in your inventory. These minipods contain random loot from related chests and pods. Altas use them for fast compact transfer, thus everything inside is packed tightly. You won't be able to put things back if you open it.
