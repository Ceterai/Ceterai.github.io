# Crystal Flower Pot

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/objects/alta/special/tools/pots/calin/icon.png" alt="Crystal Flower Pot icon" loading="lazy" width="auto" height="16px"> **Crystal Flower Pot** is a common storage unit.

A simple crystal see-through flower pot for flowers that like sunlight on their roots.  
Can be used for capturing plant critters for collection and to have them as pets.
