# Alta Terrene Display

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/objects/alta/special/tools/displays/terrene/icon.png" alt="Alta Terrene Display icon" loading="lazy" width="auto" height="16px"/> **Alta Terrene Display** is a common decorative object.

An aquarium with enough space for a small squid, or a big jello.  
Can be used for capturing water critters for collection and to have them as pets.
