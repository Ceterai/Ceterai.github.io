# Warped Cocktail

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/items/generic/food/tier4/ct_warped_cocktail.png" alt="Warped Cocktail icon" loading="lazy" width="auto" height="16px"/> **Warped Cocktail** is a legendary drink.

Despite its name and looks, this drink isn't actually made out of poisonous warped roots. Duh.  
It's a calin dish, where different crystal extracts were mixed together to create a crimson luminescent cocktail.
