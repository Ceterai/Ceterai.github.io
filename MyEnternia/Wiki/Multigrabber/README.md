# Multigrabber ★

<img src="https://raw.githubusercontent.com/Ceterai/Enternia/main/items/active/alta/tools/grab/grabber/icon_upg.png" alt="Multigrabber ★ icon" loading="lazy" height="16px" width="auto" /> **Multigrabber ★** is a legendary two-handed tool.
